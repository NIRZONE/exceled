üöÄ Getting Started with Excel

What is Excel?

Microsoft Excel is a powerful spreadsheet application used for data organization, analysis, calculation, and visualization. It's widely used in business, finance, education, and personal productivity.

Basic Terminology

Workbook

An Excel file containing one or more worksheets

Worksheet

A single spreadsheet page within a workbook

Cell

The intersection of a row and column (e.g., A1, B5)

Range

A group of cells (e.g., A1:C10)

Formula

An equation that performs calculations

Function

Pre-built formulas for common tasks

Creating Your First Workbook

  1. Open Excel application
  2. Click "Blank Workbook" or press Ctrl+N
  3. You'll see a grid of cells labeled with columns (A, B, C...) and rows (1, 2, 3...)
  4. Click any cell to select it
  5. Type data and press Enter to move to the next cell

Saving Your Work

Save As: Ctrl+Shift+S or F12
Quick Save: Ctrl+S
File Formats: .xlsx (default), .xlsm (with macros), .csv (comma-separated)

Located at the bottom, allows switching between worksheets.

üîÑ Pivot Tables

What are Pivot Tables?

Powerful tools for summarizing, analyzing, and presenting large amounts of data. They allow you to reorganize and summarize selected columns and rows to obtain desired reports.

Creating a Pivot Table

  1. Click anywhere in your data range
  2. Insert tab ‚Üí PivotTable (or Alt+N+V)
  3. Choose data range and location
  4. Drag fields to areas:
    • Rows: Categories down the left
    • Columns: Categories across the top
    • Values: Numbers to summarize
    • Filters: Filter entire pivot table

Pivot Table Components

Row Labels

  • Primary grouping dimension
  • Can have multiple levels (nested)
  • Example: Region ‚Üí State ‚Üí City

Column Labels

  • Secondary grouping dimension
  • Creates cross-tabulation
  • Example: Months across columns

Values Area

  • Numeric data to summarize
  • Multiple calculations possible
  • Can show as: Sum, Count, Average, Max, Min, etc.

Filters

  • Filter entire pivot table
  • Multiple filters allowed
  • Acts like report filter

Value Field Settings

Summarize By

  • Sum (default for numbers)
  • Count (default for text)
  • Average
  • Max / Min
  • Product
  • Count Numbers
  • StdDev / Variance

Show Values As

  • % of Grand Total: Percentage of all values
  • % of Column/Row Total: Percentage within group
  • % of Parent: Percentage of parent item
  • Difference From: Compare to base value
  • % Difference From: Percentage change from base
  • Running Total: Cumulative sum
  • % Running Total: Cumulative percentage
  • Rank: Numerical ranking
  • Index: Relative importance

Grouping Data

  • Group by: Seconds, Minutes, Hours, Days, Months, Quarters, Years
  • Can group by multiple time periods simultaneously

What-If Analysis

Find the input value needed to achieve a specific goal

  • Data tab ‚Üí What-If Analysis ‚Üí Goal Seek
  • Set cell: Formula cell with target result
  • To value: Desired result
  • By changing cell: Input cell to adjust

Example: Loan Payment

What interest rate gives a monthly payment of $1,000?

Set cell: Payment formula | To value: 1000 | By changing: Interest rate cell

Data Tables

Show how changing variables affects formula results

One-Variable Data Table:

  • Test one input variable
  • Results in rows or columns
  • Example: Loan payments at different interest rates

Two-Variable Data Table:

  • Test two input variables
  • Results in table format
  • Example: Payments varying by rate AND term

Scenario Manager

Save and compare different sets of input values

  • Data tab ‚Üí What-If Analysis ‚Üí Scenario Manager
  • Create multiple scenarios (Best Case, Worst Case, etc.)
  • Specify changing cells for each scenario
  • Create summary report comparing scenarios

Solver

Find optimal value by changing multiple variables with constraints

Solver Components

  • Objective: Cell to maximize, minimize, or set to value
  • Variable Cells: Cells that can change
  • Constraints: Limitations on variables or results
  • Solving Method: GRG Nonlinear, Simplex LP, Evolutionary

Common Uses

  • Resource allocation
  • Production planning
  • Budget optimization
  • Portfolio optimization
  • Scheduling
Note: Solver is an add-in. Enable it: File ‚Üí Options ‚Üí Add-ins ‚Üí Go (Excel Add-ins) ‚Üí Check Solver Add-in

Array Formulas (Legacy)

Formulas that perform multiple calculations on arrays

Entering Array Formulas

  • Type formula
  • Press Ctrl+Shift+Enter (not just Enter)
  • Excel adds curly braces { } around formula
  • Don't type braces manually

What are Macros?

Recorded or written sequences of commands that automate repetitive tasks

  4. Click OK

Recording Macros

Steps to Record

  1. Developer tab ‚Üí Record Macro
  2. Give macro a name (no spaces)
  3. Optionally assign shortcut key (Ctrl + letter)
  4. Choose where to store:

  5. Add description
  6. Click OK

  • Use keyboard shortcuts for cleaner code

  • Or use assigned shortcut key
  • Or assign to button/shape

Macro Security

Control whether macros can run

Security Levels

  • Disable all macros without notification: All macros blocked
  • Disable all macros with notification: Macros disabled, can enable manually
  • Disable all macros except digitally signed: Only trusted macros
  • Enable all macros: All macros run (not recommended)

Loop Through Rows

Sub ProcessRows()
    Dim lastRow As Long
    Dim i As Long
    lastRow = Cells(Rows.Count, 1).End(xlUp).Row
    For i = 2 To lastRow
        Cells(i, 3).Value = Cells(i, 1).Value * Cells(i, 2).Value
    Next i
End Sub

Copy Data Between Sheets

Sub CopyData()
    Worksheets("Sheet1").Range("A1:B10").Copy _
    Destination:=Worksheets("Sheet2").Range("A1")
End Sub

Delete Empty Rows

Sub DeleteEmptyRows()
    Dim i As Long
    For i = Cells(Rows.Count, 1).End(xlUp).Row To 1 Step -1
        If Application.WorksheetFunction.CountA(Rows(i)) = 0 Then
            Rows(i).Delete
        End If
    Next i
End Sub

Common Mistakes to Avoid

  • Not locking formulas (use $ for absolute references)
  • Hardcoding values instead of referencing cells
  • Merging cells (use Center Across Selection instead)
  • Not protecting sheets with formulas
  • Using full column references (A:A) when not needed
  • Ignoring error messages
  • Not backing up important files
  • Inconsistent data entry (dates, numbers)
  • Too many volatile functions (NOW, RAND, OFFSET)
  • Not documenting your work
